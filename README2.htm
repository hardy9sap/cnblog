三、启动方式

1. 进入项目文件根目录

2. MySQL导入数据库

        ① Shift + 右击 选择“在此处代开命令窗口”

        ② 输入mysql -u root -p 进入本地数据库管理系统

        ③ 创建一个数据库 db （create database cnblog;）

        ④ 进入db （use cnblog;）

        ⑤ 执行source db_cnblog_bak.sql

3. 使用PyCharm

        ① 打开项目

    ② Run -> Edit configurations -> 配置端口为8080

            （CMD：python3 manage.py runserver 8080）

4. 使用浏览器访问http://127.0.0.1:8080/

六、常见问题

1. 滑动验证码：

   当遇到“滑动验证码校验成功后，程序不会跳转到首页”问题时，尝试刷新页面再次登录（可能是由于自己引入的geetest模块和编写的JS代码某些地方衔接不是太好；也可能是自身Surface平板设备配置不高，调试时资源加载缓慢，才导致这样的问题。）

2. 文本验证码：

   实验时，由于验证码的噪点或噪线或字体导致验证码多次输入不正确，可以尝试注释utils工具包中的check_code.py中的71 C 75、79 C 86、88 C 92行，打开70和95行，验证码永远为123
